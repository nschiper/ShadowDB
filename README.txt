INSTALLATION
------------

CONFIGURATION
-------------

Configuring ShadowDB is done by editing file conf_aneris.emlc. In the configuration files, databases identifiers must start with "database" and the first database must have identifier "database1".

TESTING
-------

ShadowDB can be tested with the provided scripts startAneris.sh, startDbServer.sh, and startBankingClient.sh. Each of one of these scripts takes parameters whose semantics are defined in the respective script. Additionally, Aneris can be tested with script startTobcastClient.sh.
